## How to run
Open the folder in IntelliJ and update the path for JavaFX JDK, SDK, and JDBC.
Press Run. Users must login to access the appointment main screen and other application pages.
Appointment views can be changed among Default, Month, and week views using radio buttons.
Add Appointment, Update appointment, and Customer views can be accessed via Appointment views.
Customer view offers options to visit Appointment view, Add and Update customer views, and Report view.

## Additional Report
The additional report displays a list of appointment ID, start, and end datetime grouped by each each User.
