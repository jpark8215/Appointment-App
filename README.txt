
## C195 Software II
The application supports managing appointments for customers 
by providing add, update, and delete appointment and customer functions.

## JDK Version
- Java SE 11.0.11

## JavaFX Version
- JavaFX SDK 11.0.2

## How to run
Users must login to access the appointment main screen and other application pages.
Appointment views can be changed among Default, Month, and week views using radio buttons.
Customer view can be accessed via Appointment views as well as Add and Update appointment views.
Customer view offers options to go to Appointment view, Add and Update customer views, and Report view.

## Additional Report
- Displays appointment id, start, and end datetime by User
